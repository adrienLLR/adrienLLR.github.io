Headline generation

Construction of a Seq2Seq model that takes news bodies texts and generate their titles. The task is Text Summatization. Encoder and Decoder are LSTMs, embedding is done with Word2Vec and training with Teacher Forcing algorithm. Inference is then done with Greedy Decoding.

Random Act Of Pizza Classification

Classification on a dataset containing text and numerical features. Exploratory Data Analysis. Use of TF-IDF and NMF to extract relevant topics. Feature engineering comprising text features embedding into vectors thanks to wiki-words World2Vec model ( skipgram version ), narrative and time features addition. Use of Normalization, PCA, Cross Validation. Comparison between different models : Logistic Regression, Naive Bayes, Gradient Boosting, K-Nearest Neighbors. Evaluation performed with ROC AUC, accuracy and confusion matrix.

Java's custom language compiler

Homemade compiler for a homemade language called Deca which is a sub language of Java. The compiler is coded in Java and the assembly language is that of a homemade abstract machine called Ima. Use of ANTLR to do the lexical and syntactic analysis and to help build the abstract tree. It was a group project, I focused on the tree decoration, helped on the assembly language generation and I conducted the extension part which was to implement a linker to be able to generate and link object files.
